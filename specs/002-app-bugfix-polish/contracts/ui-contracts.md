# UI Contracts: 应用精简与统一化改造

## 1. GameStartScreen 组件契约

### Props

```typescript
interface GameStartScreenProps {
  mode: TrainingMode;
  title: string;
  description: string;
  onStart: () => void;
}
```

### 行为

| 条件 | 行为 |
|------|------|
| 首次进入 (ruleDismissed[mode] === false) | 自动弹出规则弹窗 |
| 再次进入 (ruleDismissed[mode] === true) | 不弹出规则弹窗 |
| 用户勾选"不再提示"并关闭弹窗 | ruleDismissed[mode] 设为 true |
| 用户点击"展示规则"按钮 | 弹出规则弹窗（始终可用） |
| 用户点击"开始训练"按钮 | 调用 onStart() |

### UI 结构

```
┌─────────────────────────┐
│      [游戏标题]          │
│    [简短描述文字]        │
│                         │
│   [展示规则] 按钮        │
│                         │
│   [▶ 开始训练] 主按钮   │
└─────────────────────────┘
```

### 规则弹窗 (Dialog)

```
┌─────────────────────────┐
│     游戏规则             │
│  ─────────────────────  │
│  [目标说明]              │
│  [玩法步骤]              │
│  [评分规则]              │
│                         │
│  ☐ 不再提示              │
│                         │
│      [知道了]            │
└─────────────────────────┘
```

## 2. Stats 页面契约

### 显示维度

| 指标 | 单位 | 说明 |
|------|------|------|
| 总训练次数 | 次 | COUNT(records) |
| 平均分数 | 分 | AVG(score), 0-100 |
| 平均准确率 | % | AVG(accuracy), 0-100 |
| 连续训练 | 天 | 连续有记录的天数 |
| 今日目标 | 次 | completed/target |
| 各模式统计 | 次 | per-mode sessions count |

### 不显示的维度

- ~~总训练时长~~ (分钟)
- ~~各模式训练时长~~ (分钟)
- ~~每分钟进度~~
